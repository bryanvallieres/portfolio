Application mobile "Intelligence artificielle"

J'ai réalisé ce projet web en tant que project lors d'un cours de developper web. L'objectif? Permettre à l'utilisateurs
de gagner en visibilité et de proposer son menu de pizzas au personne qui visite le site web. Ce site est fonctionnel pour des utilisateurs
qui n'ont pas nécéssairement les connaissances des nouvelles technologies. Il est adapté pour modifier facilement
son menu en ligne grâce à une interface d'administration sur-mesure que j'ai réalisé. Ce project est simple mais fonctionnel.
Technologies utilisées : HTML, CSS, DJANGO, PYTHON et HEROKU
